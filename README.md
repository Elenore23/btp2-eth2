# Relay System for BTP 2.0

## Introduction

This is a `relay` implementation for BTP 2.0 protocol.

### Target chains
* ICON (BTP Block)
* Ethereum 2.0 (Beacon chain)

## Getting Started
1. `git clone https://github.com/icon-project/btp2-eth2.git --recurse-submodules`
2. `cd btp2-eth2`
3. `make relay`
